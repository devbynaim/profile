Overview

Sopsync is a dynamic web application that underscores my expertise in React.js and core front-end technologies. The primary objective of this project is to emphasize my proficiency in React's core capabilities, including state management and the Context API, by intentionally choosing not to use libraries like Redux. Sopsync serves as a practical and versatile shopping list application, highlighting my commitment to core React development.

Key Features:

  1. React Core Focus: Sopsync is a testament to my mastery of React's fundamental state management and the Context API, deliberately eschewing libraries like Redux. It demonstrates my ability to harness React's intrinsic capabilities for building robust web applications.
  2. List Management System: Sopsync offers users the ability to create, manage, and organize shopping lists effortlessly. Users can create new lists, save them to local storage, and delete lists when they're no longer needed.
  3. List Status Control: Users can easily change the status of their shopping lists, marking them as "Running" while actively shopping or "Complete" once they've finished their shopping.
  4. Customizable Items: Sopsync empowers users to customize their shopping experience. They can select items from predefined categories and create custom items with quantities and prices, tailoring their lists to their specific needs.
  5. Filtering by Status: Users can conveniently filter their lists by status, allowing them to focus on "Complete," "Running," or all lists, enhancing organization and efficiency.
  6. Invoice Printing: The application includes a feature to generate and print invoices, enhancing the shopping experience by providing a clear overview of items and costs.

Why I Chose This Project::

I selected the Sopsync shopping list app as a showcase project for several reasons:

  1. React Core Focus: My deliberate choice to forgo Redux and other libraries underscores my dedication to mastering React's core capabilities and state management.
  2. Practical Application: Sopsync addresses a real-world need by simplifying the shopping process. It demonstrates my ability to develop functional and user-friendly web applications that can improve daily tasks.
  3. Versatility: The project's versatility, with features like custom item creation, status filtering, and invoice printing, highlights my capacity to create feature-rich and adaptable applications.

Challenges:

Despite my experience and skills, the Sopsync project did present some challenges:

  1. Complex UI: While my primary focus was on functionality, designing an aesthetically pleasing user interface posed a challenge. It pushed me to enhance my UI/UX design skills.
  2. Local Storage: Implementing local storage for saving lists required careful consideration of data persistence and handling edge cases, contributing to my proficiency in data management.
