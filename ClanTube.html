Overview

CleanTube is a web application that aims to provide users with a distraction-free learning experience. It allows users to create personalized playlists of educational YouTube content. With ClanTube, users can add playlists from YouTube using either the playlist ID or URL. They can also bookmark their favorite playlists, take notes, and access video descriptions - all within a clean and user-friendly interface. This project utilizes the YouTube Data API to seamlessly integrate YouTube video data and enhance the learning experience.

Functional Requirements:

  1. Adding Playlists: Users can add new playlists by providing a valid YouTube playlist URL or playlist ID. Display added playlists in the user's playlist collection.
  2. Deleting Playlists: Users can delete playlists from their collection.
  3. Adding Playlist to Favorites: Users can remove playlists from their favorites.
  4. Automatic Addition to "Recently Watched" Playlist:: Playlists that users watch are automatically added to the "Recently Watched" playlist.
  5. Note-Taking: Users can take notes while watching a video. Edit and delete notes for a personalized learning experience.

State Management Redux Toolkit:

  1. Playlists Slice: Asynchronous actions for adding, deleting, adding to favorites, and removing from favorites.
  2. Recent Playlists Slice: Actions for adding to the "Recently Watched" playlist.
  3. Notes Slice: Actions for adding, updating, and deleting notes.

Technologies Used:

  1. React.js: Foundation for building dynamic and interactive user interfaces with reusable components.
  2. Redux Toolkit: Efficient state management for playlists, favorites, notes, and critical application data.
  3. YouTube Data API: Seamless integration to search for videos, retrieve details, and organize playlists.
  4. react-rte (Rich Text Editor): Enhances note-taking functionality with rich text formatting.

Challenges Overcome:

  1. Redux Toolkit Implementation: Successful implementation of Redux Toolkit for effective state management.
  2. react-rte Implementation: Successful incorporation and optimization of the react-rte library for enhanced note-taking.
